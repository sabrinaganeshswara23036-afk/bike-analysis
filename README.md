# Bike Sharing Dashboard 🚲

Dashboard ini dibuat menggunakan Streamlit untuk menganalisis data penyewaan sepeda berdasarkan waktu, kondisi cuaca, dan pola pengguna.

## Fitur
- Filter berdasarkan tahun dan kondisi cuaca
- Menampilkan statistik penyewaan (rata-rata, maksimum, minimum)
- Analisis tren penyewaan dari tahun 2011 sampai 2012
- Analisis berdasarkan kondisi cuaca
- Analisis hari kerja dan hari libur
- clustering dengan grouping

## Dataset
Dataset yang digunakan:
- day_clean.csv → data penyewaan harian
- hour_clean.csv → data penyewaan per jam

## Cara Menjalankan

1. Install library:
pip install streamlit pandas matplotlib

2. Jalankan aplikasi:
streamlit run app.py
